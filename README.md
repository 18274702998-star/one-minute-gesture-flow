# One-Minute Gesture Flow

A React + Vite mini game for a one-minute hand gesture exercise session.

## Tech Stack

- React
- Vite
- MediaPipe Tasks Vision
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite in your browser.

## Build

```bash
npm run build
```

## Project Structure

```text
src/
  App.jsx
  App.css
  CoachUiPrototype.jsx
  CoachUiPrototype.css
  main.jsx
```
